@dotdo/events - Event streaming, CDC, and lakehouse analytics for Durable Objects

A comprehensive event streaming library for Cloudflare Durable Objects providing:

  • Event Batching: Efficient batched emission to events.do or custom endpoints
  • Reliable Delivery: Alarm-based retries with exponential backoff and circuit breaker
  • CDC (Change Data Capture): Automatic change tracking for collections
  • PITR (Point-in-time Recovery): SQLite bookmark capture for time-travel queries
  • R2 Lakehouse: Stream events to R2 for DuckDB analytics
  • Browser SDK: Lightweight client-side analytics tracking
  • Webhook Verification: Signature verification for GitHub, Stripe, WorkOS, Slack, Linear, Svix
  • Pub/Sub: Event subscriptions with pattern matching and delivery guarantees

Example: Basic Usage

import { EventEmitter, CDCCollection } from '@dotdo/events'

export class MyDO extends DurableObject {
  events = new EventEmitter(this.ctx, this.env, { cdc: true })
  users = new CDCCollection(this.collection('users'), this.events, 'users')

  async createUser(user: User) {
    this.users.put(user.id, user) // Automatically emits CDC event
  }
}

Example: Browser Analytics

import { init, track, page } from '@dotdo/events/browser'

init({ endpoint: '/api/events' })
page() // Track page view
track('button_click', { button: 'signup' })
