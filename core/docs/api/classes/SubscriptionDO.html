Class SubscriptionDO

Durable Object for managing event subscriptions and delivery.

SubscriptionDO provides a complete pub/sub system with:

  • Pattern-based subscriptions (glob wildcards)
  • Automatic retry with exponential backoff
  • Dead letter queue for failed deliveries
  • Full delivery audit logging
