Class EventsSDK

Browser analytics SDK class for event tracking.

The EventsSDK provides a full-featured client for tracking page views, custom events, and user identification with automatic batching and retry.

Example

const sdk = new EventsSDK({
  endpoint: '/api/events',
  batchSize: 10,
  flushInterval: 5000,
  maxRetries: 3
})

sdk.page()
sdk.track('purchase', { amount: 99.99, currency: 'USD' })
sdk.identify('user-123', { name: 'Alice' })

// Clean up when done
sdk.destroy()
