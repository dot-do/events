Class EventEmitter

Lightweight event emitter for Durable Objects with batching, retries, and R2 streaming.

The EventEmitter provides reliable event delivery with the following features:

  • Batching: Events are batched together for efficient transmission
  • Automatic Retries: Failed deliveries are retried using alarm-based exponential backoff
  • Circuit Breaker: Protects against cascading failures when the endpoint is down
  • CDC Integration: Automatic capture of SQLite bookmarks for point-in-time recovery
  • R2 Streaming: Optional streaming to R2 for lakehouse analytics
  • Hibernation Support: Batch persistence across DO hibernation cycles

Example: Basic Usage

import { EventEmitter } from '@dotdo/events'

export class MyDO extends DurableObject {
  events = new EventEmitter(this.ctx, this.env, {
    cdc: true,
    r2Bucket: this.env.EVENTS_BUCKET
  })

  async doSomething() {
    this.events.emit({ type: 'custom.event', data: { foo: 'bar' } })
  }

  async alarm() {
    await this.events.handleAlarm()
  }
}

Example: With Custom Endpoint and API Key

const events = new EventEmitter(ctx, env, {
  endpoint: 'https://my-events.example.com/ingest',
  apiKey: env.EVENTS_API_KEY,
  batchSize: 50,
  flushIntervalMs: 2000
})

Example: Handling Circuit Breaker

try {
  events.emit({ type: 'custom.event' })
} catch (e) {
  if (e instanceof CircuitBreakerOpenError) {
    console.log(`Circuit breaker open, will reset at ${e.resetAt}`)
  }
}
